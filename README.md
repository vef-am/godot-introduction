This repository contains my practice project files from the **Godot 101 – Game Engine Foundations** course by **Zenva**.

It is a personal learning workspace where I explored core Godot concepts such as:

- Scene and node structure
- Scripting with GDScript
- Materials and basic visuals
- 2D/3D scene setup
- Parenting and movement fundamentals

## Project Info

- **Engine:** Godot 4.6 (project feature tag)
- **Main scene:** `res://Scenes/main.tscn`
- **Project name:** `GodotIntroduction`

## Folder Structure

- **Scenes/**: Practice scenes for lessons (`main`, `lighting`, `parenting`, etc.)
- **Scripts/**: GDScript files used in course exercises
- **Materials/**: `.tres` material resources
- **Sprites/**: Imported textures/sprites

## How to Open

1. Install **Godot 4.x** (recommended: latest stable 4.x).
2. Open Godot and click **Import**.
3. Select this folder (the one containing `project.godot`).
4. Run the project or open individual scenes in `Scenes/`.
